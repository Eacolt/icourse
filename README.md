

###########介绍
icourse-cli, 一个基于vue.js和pixi.js的Cli;

###########部署步骤
1. npm i -g icourse
2. 在你电脑全局node_modules下,cd icourse => 安装依赖 npm i;
3. 进入你要搭建webApp的地方
4. icourse -v (show me the version)
5. icourse init myapp (create your webapp);
6. cd myapp => npm i (finished);
7. npm run serve
8. npm run build
